### abs

##### Description

Return the absolute value of a number. The argument may be a plain or long integer or a floating point number. If the argument is a complex number, its magnitude is returned.

##### Syntax
```
abs(x)
```

##### Example

```python
>>> a = -1
>>> abs(a)
1
```
